# ➕➖ Simple Calculator ✖️➗

*A calculator so simple, it makes your phone's calculator app feel overengineered.*

Built with nothing but Python and pure `tkinter` stubbornness, this little desktop app adds, subtracts, multiplies, and divides — and does absolutely nothing else. No history, no scientific mode, no dark theme. Just numbers doing number things.

## 🧮 What it does

- Adds two numbers, because math
- Subtracts two numbers, because sometimes you have too much
- Multiplies two numbers, because addition is slow
- Divides two numbers (results may be judgmental if you divide by zero)
- Has a big satisfying `CLR` button for when you regret your life choices

## 🖼️ What it looks like

A classic number pad (0–9), four operator buttons, an equals sign, and a clear button — all sitting in a plain `tkinter` window that looks like it's from 2009 and is proud of it.

## 🛠️ Tech stack

- **Python** 🐍
- **tkinter** (yes, the one built into Python — no extra installs, no drama)

## 🚀 Getting started

```bash
git clone https://github.com/mishtee-khanna/simple-calculator.git
cd simple-calculator
python calculator.py
```

That's it. No `pip install`, no `.env` file, no 47-step setup guide. Just run it.

## ⚠️ Known quirks

- Only works with whole numbers — decimals will politely crash the app
- One operation at a time — it doesn't do order of operations, it doesn't even know what that is
- Dividing by zero will not be gentle with you

## 💡 Why does this exist?

Because every developer needs to build a calculator at some point. It's basically a rite of passage, like breaking your `main` branch or Googling "how to center a div."
